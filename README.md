# Event_Management_System
# 🔹 Event Management System – Django Web Application

##  Description

The Event Management System is a full-stack web application built using Django that enables users to explore, book, and manage events in real-time. It provides a centralized platform for handling different types of events such as workshops, concerts, and social gatherings.

The frontend is developed using HTML and CSS to ensure a simple and user-friendly interface, while Django manages backend operations including authentication, business logic, and data handling. MySQL is used as the database to store user details, event information, and booking records securely.

---

##  Features

* User registration and login
* Browse and search events
* Real-time event booking
* Admin panel for event management
* Booking history and tracking

---

##  Technologies Used

* Backend: Python, Django
* Frontend: HTML, CSS
* Database: MySQL

---

##  How to Run

1. Clone the repository
2. Install dependencies (`pip install -r requirements.txt`)
3. Configure MySQL database
4. Run server (`python manage.py runserver`)

---

##  Conclusion

This project demonstrates full-stack development using Django with real-time booking functionality and efficient database management.
